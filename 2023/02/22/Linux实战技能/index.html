Linux实战技能100讲

万能的帮助命令

为什么要学帮助命令

  • Linux的命令基本操作方式是命令行
  • 海量的命令不适合死记硬背
  • 要升级大脑

man 帮助

  • man是manual的缩写
  • man帮助用法演示
    • man ls
  • man也是一条命令，分9章，可以使用man命令获得man的帮助
    • man 7 man

help帮助

  • shell（命令解释器）自带的命令成为内部命令，其他的是外部命令

  • 内部命令使用help帮助

    • help cd

  • 外部命令使用help帮助

    • ls --help

info帮助

  • info帮助比help更详细，作为help的补充
    • info ls

使用网络资源（所搜引擎和官方文档）

  • 解放大脑，使用所搜引擎
  • 使用官方文档，获取最新的特性

强大的文本编辑器vi

多模式文本编辑器

  • 多模式产生的原因
  • 四种模式
    • 正常模式(Normal-mode)
    • 插入模式(Insert-mode)
    • 命令模式(Command-mode)
    • 可视模式(Visual-mode)

正常模式

  • 进去其他模式转换命令

    • 进入插入模式

      i 光标当前的位置进入到插入模式

      I 光标所在行的开头进入插入模式

      a 光标后面的位置进入到插入模式

      A 光标所在行的结尾进去插入模式

      o 光标所在行的下行进去插入模式

      O 光标所在行的上行进去插入模式

    • 进入可视化模式 v V ctrl + v

    • 进去命令模式 :

    • 从其他模式回到正常模式 esc

  • 文本的操作

    • 光标的移动

      h 向左移动

      j 向下移动

      k 向上移动

      l 向右移动

    • 基本操作

      y 复制

      yy 单行复制

      nyy 多行复制

      y + $ 从光标位置复制到行尾部

      d 剪切

      dd 剪切光标所在的一整行

      ndd 多行剪切

      d + $ 从光标位置剪切到行尾

      p 粘贴

      u 撤销

      ctrl + r 重做撤销的那步命令

      x 单个字符的删除

      r 单个字符的替换

      :set nu 显示当前所在行号

      n + G 快速的切换到第n行

      g 文本的第一行

      G 文本的最后一行

      ^ 光标所在行的行首

      $ 光标所在行的行尾

命令模式

  • 基本操作

    • :w 写入

    • :q 退出

    • :! 执行Shell命令

    • :s 替换

      :s/old/new 用新的替换旧的默认是光标所在的那一行的第一个旧的

      :%s/old/new 替换整个文件每一行的第一个旧的那个

      :%s/old/new/g 替换整个文件中的旧的

      :n1,n2s/old/new/g 替换n1到n2行之间的旧的

    • / 查找

      n 下一个查找到的元素

      Shift + n 上一个查找到的元素

    • set 设置命令

      :set nu 显示行号

      :set nonu 不显示行号

      写在配置文件中vimrc去

可视模式

  • 三种进入可视模式的方式

    • v 字符可视模式

    • V 行可视模式

    • ctrl + v 块可视模式

      配合d和I 命令可以进行块的便利操作

用户与权限管理

  • 多用户操作系统的目的是隔离

    ​ 用户权限隔离

    ​ 系统资源隔离

    ​ root用户与普通用户的区别

多用户操作系统

  • 用户权限隔离

  • 系统资源隔离

  • root用户与普通用户的区别

用户管理常用命令

  • useradd 新建用户

  • userdel 删除用户

  • passwd 修改用户密码

  • usermod 修改用户属性

    usermod -d home_dir username 把用户username家目录换到新的位置

    usermod -g groupname username 把用户username的用户组修改为groupname

  • chage 修改用户属性

组管理命令

  • groupadd 新建用户组

    groupadd newgroup 创建新的用户组

    usermod -g newgroup username把用户username的用户组修改为newgroup

    useradd -g newgroup username 新建用的时候直接指定了用户组为newgroup

  • groupdel 删除用户组

用户配置文件

/root root用户的家目录

/home/USERNAME 普通用户默认家目录位置

/etc/passwd 用户配置文件

/etc/shadow 用户密码相关配置未见

/etc/group 用户组配置文件

用户切换

  • su 切换用户

    su - USERNAME 使用login shell方式切换用户

  • sudo 以其他用户身份执行命令

    visudo 设置需要使用sudo的用户（组）

文件权限的表示方法

  • 字符权限表示方法

    r 读

    w 写

    x 执行

  • 数字权限的表示方法

    r = 4

    w = 2

    x = 1

  • -rw- r-x r-- 1 username groupname mtime filename

    rw- 文件属主的权限

    r-x 文件属组的权限

    r-- 其他用户的权限

    创建文件有默认权限，根据umask值计算

目录权限的表示方法

  • x 进入目录

  • rx 显示目录内的文件名

  • wx 修改目录内的文件名

修改权限命令

chmod 修改文件、目录权限

chown 更改属组、属主

chgrp 可以单独更改属组，不常用

###特殊权限

SUID 用户二进制可执行文件，执行命令时取得文件属主权限

SGID 用于目录，在该目录下创建新的文件和目录，权限自动更改为该目录的属组

SBIT 用于目录，该目录下新建的文件和目录，仅root和自己可以删除
